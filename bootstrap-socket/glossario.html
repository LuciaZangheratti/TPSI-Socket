Glossario dei Termini

Definizioni e spiegazioni dei concetti utilizzati nel nostro sito

Socket

Teoria software progettata per la trasmissione e la ricezione di dati attraverso una rete, per inviare e ricevere dati tra un client e un server.

TCP/IP

Un protocollo di comunicazione che consente il trasferimento affidabile di dati tra dispositivi connessi alla rete.

HTTP

Protocollo di trasferimento ipertestuale utilizzato per comunicare tra client (come i browser) e server web.

UDP

Un protocollo di comunicazione senza connessione che offre una trasmissione veloce ma senza garanzie di affidabilità.

VoIP

La VoIP (Voice over IP) è una tecnologia che permette la trasmissione di voce e video attraverso la rete Internet.

Chat Room Multiutente

Area virtuale in cui si svolgono le comunicazioni in tempo reale fra due o più utenti collegati allo stesso canale telematico.

Threading

Suddivisione di un processo in due o più sottoprocessi(istanze).

File Server

E' un server centrale, che permette ai client connessi in una rete di computer di accedere ai file archiviati su disco.

Ping personalizzato

Applicazione personalizzata che simula il comando ping per verificare la connessione con un dispositivo remoto.

AF-INET

Famiglia di indirizzi che fornisce la comunicazione tra processi che vengono eseguiti sullo stesso sistema o su sistemi differenti.

SOCK_STREAM

Tipologia di socket che fornisce una connessione sequenziale, affidabile e full-duplex.

SOCK_DGRAM

Tipologia di socket che supporta l’invio di datagrammi, ovvero pacchetti di dati indipendenti e inaffidabili.

Messaggi di SYN e ACK

Il flag SYN indica l'accordo del server a stabilire una connessione e il flag ACK conferma la ricezione del pacchetto SYN del client.

Email (SMTP/POP3/IMAP)
  • SMTP:gestisce la trasmissione delle email tra i server
  • POP3:scarica le email su un dispositivo
  • IMAP:sincronizza le email tra più dispositivi
FTP (File Transfer Protocol)

Protocollo usato per trasferire file tra computer su Internet.
